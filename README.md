# NServiceBus.Encryption.MessageProperty

NServiceBus.Encryption.MessageProperty provides encryption for properties on messages.

It is part of the [Particular Service Platform](https://particular.net/service-platform), which includes [NServiceBus](https://particular.net/nservicebus) and tools to build, monitor, and debug distributed systems.

See the [Message Property Encryption documentation](https://docs.particular.net/nservicebus/security/property-encryption) for more details on how to use it.
